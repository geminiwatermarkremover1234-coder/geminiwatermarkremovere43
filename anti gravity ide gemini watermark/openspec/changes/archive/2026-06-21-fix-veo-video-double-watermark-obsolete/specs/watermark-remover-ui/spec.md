## MODIFIED Requirements

### Requirement: Modern Premium Dashboard UI
The interface SHALL be a single-page responsive dashboard styled with Tailwind CSS or custom CSS, using a dark theme with glassmorphic cards, smooth animations, and typography from Google Fonts. It SHALL provide a visible option to switch between Gemini Omni and Google Veo video cleaners.

#### Scenario: User opens the dashboard
- **WHEN** the user navigates to the application
- **THEN** the browser SHALL display a dark-mode styled dashboard with custom cards, layout grid, header navigation, and cleaner model toggle options
